# Firebase setup (P5)

The app bootstraps Firebase on startup when platform options are available.
Favorites sync to Firestore after login; local cache remains the offline source.

## Prerequisites

1. Firebase project with **Authentication** (email/password, Google, Apple on iOS) and **Cloud Firestore**.
2. FlutterFire CLI: `dart pub global activate flutterfire_cli`
3. From project root: `flutterfire configure`

## Dependencies (`pubspec.yaml`)

- `firebase_core`
- `firebase_auth`
- `cloud_firestore`
- `firebase_analytics` (optional events)
- `google_sign_in`
- `sign_in_with_apple` (iOS)

## Bootstrap

1. `lib/firebase_options.dart` — generated by FlutterFire CLI.
2. `lib/core/firebase/firebase_bootstrap.dart` — calls `Firebase.initializeApp`.
3. `main.dart` — `await bootstrapFirebase()` before `runApp`.

## Platform files

- **Android:** `android/app/google-services.json` (referenced by Gradle plugin).
- **iOS:** `ios/Runner/GoogleService-Info.plist` (add via `flutterfire configure`).

## Firestore path

`users/{uid}/favorites/{pokemonId}` — fields: `pokemonId` (int), `addedAt` (timestamp).

Deploy rules (requires [Firebase CLI](https://firebase.google.com/docs/cli) logged in):

```bash
firebase login
firebase deploy --only firestore:rules
```

PowerShell: use quotes if passing multiple targets — `firebase deploy --only "firestore:rules"`.

The project uses `.firebaserc` (default: `pokedex-app-c5e90`) and `firebase.json` with `firestore.rules` → `docs/firestore.rules`.

## Google Sign-In (Android)

1. Add **SHA-1** and **SHA-256** in Firebase Console → Project settings → Your apps → Android.

```bash
cd android && ./gradlew signingReport
```

2. **Re-download** `google-services.json` after adding fingerprints. The file must include an `oauth_client` with `"client_type": 1` (Android). If you only see `client_type: 3`, Google Sign-In will fail.

3. The app uses the Web client ID as `serverClientId` in `FirebaseAuthConfig` (required for Firebase Auth idToken on Android).

4. Rebuild the app after replacing `google-services.json`:

```bash
flutter clean && flutter run
```

## Apple Sign-In (iOS)

Enable **Sign in with Apple** in Firebase Authentication and Xcode capabilities.

## Auth behavior

| Flow | Firebase | Mock (tests / no bootstrap) |
|------|----------|----------------------------|
| Email login/signup | `FirebaseAuth` | SharedPreferences |
| Email verification | Link + reload | 6-digit OTP |
| Password reset | `sendPasswordResetEmail` | OTP + new password |
| Change password | reauth + `updatePassword` | local password store |
| Google / Apple | OAuth credentials | Snackbar stub |
